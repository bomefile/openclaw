# 家庭保险配置方案

> 基于个人家庭结构梳理的保障思路与产品推荐
> 整理时间：2026-06-16

---

## 一、保障核心思路：先保住"印钞机"

**家庭刚性月支出：约 14,300 元**
- 房贷还款：2,800 元
- 房租：4,500 元
- 育儿开支：3,000 元
- 赡养父母 + 妹妹：约 4,000 元

**投保优先级：**
自己（重疾 + 医疗 + 意外 + 定寿）→ 父母和妹妹（医疗 + 意外）→ 孩子（重疾 + 医疗 + 意外）

**各成员配置要点：**

| 成员 | 核心思路 |
|------|----------|
| 你（38岁，经济支柱） | 四大险种配齐，重疾保额覆盖3-5年收入 |
| 父母（近65岁） | 医疗险 + 意外险，重疾险大概率保费倒挂，不建议 |
| 妹妹（20岁，上学/无收入） | 百万医疗 + 意外险，重疾择机补充 |
| 孩子（如有） | 重疾 + 医疗 + 意外，无需寿险 |

---

## 二、你的核心保障（四大险种配齐）

### 百万医疗险
- **作用：** 报销大额住院医疗费，保额数百万
- **推荐：** 太平洋蓝医保好医好药Pro版 / 星相守2号
- **特点：** 保证续保20年，社保报销后超过免赔额基本都能报销
- **年保费：** 约 300 元

### 重疾险
- **作用：** 确诊大病一次性赔钱，弥补收入损失
- **推荐：** 超级玛丽15号 / 达尔文12号
- **建议保额：** 50 万（覆盖3-5年收入）
- **年保费：** 约 6,000-8,000 元（保终身）；保至70岁约 3,000-4,000 元

### 意外险
- **推荐：** 大护甲7号旗舰版 / 小蜜蜂6号
- **建议保额：** 50-100 万
- **包含：** 意外医疗、猝死保障、住院津贴
- **年保费：** 约 300 元

### 定期寿险
- **作用：** 万一不幸身故，保障家人生活、还清债务
- **推荐：** 擎天柱12号泰然一生版 / 大麦2026A
- **建议保额：** 200-300 万
- **年保费：** 约 1,200-2,200 元

---

## 三、父母保障（近65岁）

### 医疗险
- **健康时：** 蓝医保好医好药版 / 金医保3号，保证续保20年或终身防癌，约 1,000-1,500 元/人/年
- **有基础病时：** 普惠免健告医疗险，约 500-1,000 元/人/年

### 意外险
- **推荐：** 小蜜蜂老人意外险2026版 / 大护甲7号高龄版
- 60岁年保费约 198 元；高龄版约 300-500 元

---

## 四、妹妹保障（20岁）

- **百万医疗险：** 蓝医保好医好药Pro版，约 200 元/年
- **意外险：** 大护甲7号旗舰版，约 100-150 元/年
- **重疾险（择机补充）：** 30万保额保终身，约 3,000-4,000 元/年，20岁买比30岁后便宜很多

## 六、补充提醒

1. **惠民保优先：** 建议先给每个成员配置当地惠民保（城市普惠型商业医疗险），一年100-200元，作为基础兜底
2. **社保核查：** 确认自己和家人的居民医保是否断档，先补齐社保再补充商业保险
3. **健康告知：** 如实回答，问什么答什么，不必多说不该说的
4. **长期产品慎换：** 重疾险一旦定下来不要轻易换，追新品反而白花钱
5. **核心逻辑：** 花小钱、防大风险

> 以上产品与价格基于2026年6月市场公开信息整理，实际费用以投保时各保险公司官方报价为准。
