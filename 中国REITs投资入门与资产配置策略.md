# 中国 REITs 投资基础知识与配置策略

> REITs（Real Estate Investment Trusts，不动产投资信托基金）系统入门指南

---

## 一、中国REITs是什么？

中国目前的REITs准确来说是**公募基础设施REITs**，和美国、新加坡、日本的传统房地产REITs有所区别。

### 底层资产（非住宅类）

- 高速公路
- 产业园区
- 数据中心
- 仓储物流
- 保障房
- 清洁能源
- 污水处理
- 港口码头

### 运作机制

```
投资人 → 购买REITs基金 → 持有基础设施资产 → 产生租金/通行费/运营收入 → 分红给投资者
```

---

## 二、REITs vs 股票 vs 房产

| 项目 | REITs | 股票 | 房产 |
|------|-------|------|------|
| 门槛 | 几百元 | 几百元 | 几十万起 |
| 流动性 | 高 | 高 | 低 |
| 分红 | 高 | 中 | 低 |
| 波动 | 中 | 高 | 低 |
| 管理成本 | 无 | 无 | 高 |
| 杠杆 | 无 | 无 | 可贷款 |
| 收益来源 | 分红+升值 | 企业增长 | 租金+升值 |

**本质：** 房东升级版，不需要装修、招租、收租、维修。

---

## 三、收益来源

### 现金分红

```
某高速公路REIT 分红率：5%~8%（每年分配）
```

### 价格上涨

```
发行价 4元 → 几年后 5元 → 资本收益
```

**总收益 = 分红收益 + 价格上涨**

---

## 四、中国REITs主要分类

| 类别 | 代表产品 | 特点 |
|------|---------|------|
| **高速公路REITs** | 华夏中国交建REIT、平安广河REIT | 现金流稳定，分红稳定，类似"收过路费" |
| **产业园REITs** | 博时蛇口产园REIT、东吴苏园产业REIT | 受经济周期影响，类似"收厂房租金" |
| **仓储物流REITs** | 红土创新盐田港REIT | 电商驱动，增长性较好 |
| **保障房REITs** | 华夏北京保障房REIT | 政策支持，波动较小 |
| **能源REITs** | 中航首钢绿能REIT | 分红较高 |

---

## 五、核心投资指标

### 1. 分红率

```
年分红 0.3元 ÷ 当前价格 4元 = 股息率 7.5%
```

### 2. P/NAV

类似房地产估值：价格 ÷ 净资产价值，越高说明越贵。

### 3. 出租率

```
95% → 优于 → 70%
```

### 4. 现金流覆盖率

衡量未来分红能力。

## 六、投资策略对比

### 策略一：纯收息策略

适合 35岁以上、希望有被动收入的人群。

```
配置：70% REITs + 30% 货币基金
目标：年化 5%~8%
特点：波动小
```

### 策略二：定投策略

```
每月 2000~5000 元，持续买入
适合：长期积累
优点：避免择时
```

### 策略三：折价策略

关注 P/NAV < 1 时买入，类似"打折买房"，安全边际高。

### 策略四：股债REITs组合

```
40% 宽基ETF + 30% REITs + 20% 黄金 + 10% 现金
→ 风险明显降低
```

## 七、个人配置建议（38岁/月入3万）

### 方案A：稳健型（资产增值为主）

```
50% 沪深300ETF
20% 中证500ETF
20% REITs
10% 现金
```

### 方案B：现金流增强（兼顾增长和分红）

```
40% 沪深300ETF
30% REITs
20% 纳斯达克100ETF
10% 现金
```

### 方案C：未来退休现金流（40岁后逐步调整）

```
30% 股票ETF
50% REITs
20% 现金/债券
```
利用REITs持续分红补充收入。

---

## 八、各类资产收益风险对比

| 策略 | 预期年化 | 波动 | 操作难度 |
|------|---------|------|---------|
| 银行存款 | 1%~2% | 极低 | 极低 |
| 债券基金 | 2%~4% | 低 | 低 |
| REITs收息 | 4%~8% | 中低 | 低 |
| 宽基ETF定投 | 8%~12% | 中高 | 低 |
| 个股投资 | -50%~+50% | 高 | 高 |

---

> 💡 **核心观点：** REITs 最有价值的地方不是追求暴涨，而是作为股票和现金之间的**"第三类资产"**，提升组合的现金流能力和抗波动能力。对于家庭资产规模逐渐增长、未来希望建立被动收入的人来说，通常配置 **10%~30% REITs** 是比较合理的区间。
