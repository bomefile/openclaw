# 定投策略：买与卖的复利闭环

很多人在定投中容易陷入"只买不卖"或"乱卖"的误区。复利不是靠"利滚利"自动实现的，而是靠"每次赚到的钱重新投入去赚下一笔"。

## 一、买入端

**核心原则：不是"每期买一样的钱"，而是"越跌越买，越贵少买"**

| 策略 | 做法 | 效果 |
|------|------|------|
| 定时定额 | 每月固定金额 | 入门级，自动摊平 |
| 定时不定额 | 设置基准，跌多加仓，涨多减仓 | 收益率更高，复利效果更强 |
| 估值定投 | PE/PB低于历史百分位时加倍买 | 更科学，适合指数基金 |

**实操建议：**
- 新手：定时定额就够了
- 进阶：加一个**温度计规则**，比如：
  - 沪深300 PE < 12 → 双倍买入
  - PE 12~15 → 正常买入
  - PE > 18 → 减半或暂停

## 二、卖出端 —— 大多数人缺失的环节

**不做止盈，复利就是空话。** 浮盈不兑现，回撤就把利润吃回去了。

**卖出策略（三选一即可）：**

**1. 目标收益率止盈**
- 设定目标，比如+20%或+30%
- 达到就卖出一部分（比如50%）
- 落袋的钱重新投入定投池
- 这是最直接的"复利切换"——前一笔利润变成下一笔的本金

**2. 分批止盈（更推荐）**
- 涨10% → 卖20%
- 涨20% → 卖30%
- 涨30% → 卖50%
- 越涨越卖，永远不贪最后一口

**3. 高位回落止盈（右侧止盈）**
- 不止盈在最高点，而是从最高点回落一定幅度（如-10%）时卖出
- 适合牛市行情

## 三、复利闭环的实操流程

```
每月收入
 ↓
定投买入（份额积累）
 ↓
达到止盈线
 ↓
分批卖出（落袋利润）
 ↓
利润 + 每月新资金一起再投入
 ↓
用更大的本金重复以上过程
```

**关键点：** 止盈出来的钱**不要花掉**，放回定投池。这样才能产生复利效应。

## 四、一个具体配置参考

假设每月可投入 5000 元：

```
买入端：
 每月10日自动买入
 基准金额：5000
 沪深300 PE < 11 → 买10000
 PE 11~14 → 买5000
 PE > 16 → 买2500

卖出端：
 持仓收益率+25% → 卖30%仓位
 收益率+40% → 再卖30%
 收益率+50% → 全部卖出或留底仓

利润处理：
 卖出所得全部转入货币基金
 作为后续"低位加倍买入"的弹药
```

**你不是在"炒股"，你是在经营一套系统。** 每次买卖都是系统的一个指令，不靠情绪。
