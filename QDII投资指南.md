## ⚠️ 最关键的问题：溢价

这是QDII基金最坑的地方，不知道这个会亏很多。

### 什么是溢价？

```
净值（真实值） = 底层美股的实际价值
交易价格 = 市场上大家买卖的价格

当 交易价格 > 净值 → 溢价 = 买贵了
当 交易价格 < 净值 → 折价 = 买便宜了
```

### 现实案例（常发生）

```
513100 纳指ETF
净值: 1.50元
交易价: 1.65元
溢价: 10%
```

**这意味着你一买就亏了10%**——哪怕指数不涨不跌，等溢价回归就亏了。

### 怎么避免

```
操作原则：
├─ 买之前先查「溢价率」
├─ 天天基金/券商APP搜代码 → 看实时溢价
├─ 溢价 < 1% → 正常，可以买
├─ 溢价 1-3% → 谨慎，买少点
├─ 溢价 > 3% → 别买，等回落到正常再买
└─ 溢价 > 5% → 绝对的坑，谁买谁亏
```

**QDII基金因为额度有限，经常被买到高溢价，尤其是在美股大涨的时候。**

## 🔄 复利在QDII上怎么玩

### 和沪深300ETF一样的部分

```
每月定投
├─ 设自动扣款（支付宝/天天基金/券商都可以）
├─ 分红选「红利再投资」
├─ 不看盘，不管涨跌
└─ 关键是：别停

复利公式一样的：
每月3000，年化8%，30年
总投入：108万
最终价值：约 447万
```

### 不一样的部分

| 维度 | 沪深300ETF | QDII纳指ETF |
|---|---|---|
| 历史年化 | ~6-8% | **~10-15%**（过去10年） |
| 最大回撤 | -72% | -50% |
| 溢价风险 | 基本无 | **有，且常见** |
| 汇率影响 | 无 | **人民币贬值=额外收益** |
| 额度限制 | 无 | **可能限购/暂停申购** |

### 📊 汇率因素：一个隐形Buff

QDII基金用人民币买美元资产：

```
场景：人民币贬值（过去5年趋势）
买QDII = 美股涨了 + 美元升值了 = 双重收益 ✅

场景：人民币升值
买QDII = 美股涨了 - 美元贬值 = 收益打折扣
```

过去几年，人民币从6.3贬到7.3左右，**汇率贡献了约15%的额外收益**。

## 🎯 双引擎定投组合方案

```
每月定投金额：假设5000元
├─ 60% 沪深300ETF（510300/510310） → A股打底
├─ 30% 纳指ETF（513100） → 美股科技增长
└─ 10% 标普500ETF（513500） → 美股稳健补充
```

### 为什么这样配？

| 部分 | 逻辑 |
|---|---|
| **A股60%** | 主场，较低估值，有安全边际 |
| **纳指30%** | 全球最强的科技股权重，长期增长最猛，但贵 |
| **标普10%** | 补充纳指没有的消费/医疗/金融 |

这样即使A股十年不涨，还有美股的收益兜底；反过来美股崩了，A股也可能补上。

### QDII买入前检查清单

- [ ] 查实时溢价率（券商APP/天天基金）
- [ ] 溢价 < 1% → 直接买
- [ ] 溢价 1-3% → 减量买入
- [ ] 溢价 > 3% → 等回调再买
- [ ] 确认基金未暂停申购
