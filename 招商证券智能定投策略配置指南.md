# 招商证券智能定投策略配置指南

> 指数定投理论策略 vs 券商APP实际支持的自动化策略的对应关系

---

## 一、买入策略对应关系

| 招商证券 | 实际含义 |
|---------|---------|
| 定期定投 | 每周/每月固定买入 |
| 日涨幅买 | 根据当天涨跌幅买入 |
| 反弹买 | 跌后反弹触发买入 |
| 定时委托买 | 指定时间买入 |

### 1. 定期定投

对应：**基础仓位**

```
每月15日 → 买入 3000元
```

### 2. 日涨幅买

对应：**跌幅加仓**

```
当日跌幅 ≥ 2% → 买入 2000
当日跌幅 ≥ 4% → 买入 4000
当日跌幅 ≥ 6% → 买入 6000
```

> ⚠️ 注意：这里是**当天跌幅**，不是**累计跌幅**。
>
> 举例：
> - 沪深300：昨收 4.0 → 今收 3.8（跌5%），触发买入
> - 第二天：3.8 → 3.79（跌0.3%），不触发
>
> 即使整体已经跌很多，只要当日跌幅小就不会触发。
>
> 所以：**日涨幅买 ≠ 回撤买**

### 3. 反弹买

逻辑：`左侧抄底 → 确认企稳`

```
先跌 3% → 再反弹 1% → 触发买入
```

示例：
```
-5% → -8% → -10% → -9%（反弹1%）→ 触发买入
```

**适合品种：** 创业板ETF、纳指ETF（波动大的品种）

### 4. 限价单设置

ETF 流动性好的情况下，直接设置**当前价**即可：

```
沪深300ETF 价格 4.520
卖一 4.521 / 买一 4.520

设置：4.521（当前价）
```

> ❌ 不要设置 4.500 这种远低于市价的限价，可能永远成交不了。

## 二、PE 估值无法自动触发怎么办

招商证券很多版本不支持 PE 百分位的自动触发。

**推荐外部工具查询估值：**
- [且慢估值表](https://qieman.com/idx-eval)
- [韭圈儿估值表](https://www.jiucaigongshe.com/estimate)
- [中证指数官网](https://www.csindex.com.cn)

**做法：** 看估值后人工调整定投金额
```
估值正常 → 本月买 3000
估值偏低 → 本月买 6000
```

## 三、卖出策略对应

| 招商证券 | 实际含义 |
|---------|---------|
| 定价卖 | 目标价格卖出 |
| 日涨跌幅卖 | 暴涨减仓 |
| 止盈止损卖 | 收益率止盈 |
| 回落卖 | 移动止盈 |
| 开板卖 | 涨停/跌停开板卖出 |
| 定时委托卖 | 定期再平衡 |

### 1. 定价卖

对应：**目标价格卖出**
```
买入价 4元 → 卖出价 5元 → 涨到5元自动卖
```

### 2. 日涨跌幅卖

对应：**暴涨减仓**
```
当日上涨 8% → 卖出 10%
```
适合创业板、科创50，不太适合长期指数投资。

### 3. 止盈止损卖

对应：**收益率止盈**
```
盈利 20% → 卖出 20%
盈利 40% → 卖出 30%
```
最接近标准收益率卖出策略。

### 4. 回落卖（推荐）

对应：**移动止盈**
```
ETF 最高盈利 50%
设置：回落 10% → 卖出

过程：50% → 45% → 40% → 触发卖出
```
效果最好，是最推荐的卖出策略。

### 5. 定时委托卖

对应：**定期再平衡**
```
每半年卖一次 / 每年卖一次
```

## 四、推荐配置方案

**背景：** 38岁，月入3万，有房租和家庭负担，追求长期资产积累

### 买入策略

**策略1 - 定期定投：**
```
每月 15 日 → 买入 5000 元
```

**策略2 - 日跌幅加仓：**
```
跌 2% → 加仓 2000
跌 4% → 加仓 4000
跌 6% → 加仓 6000
```

### 卖出策略

**只启用：回落卖**
```
盈利超过 30% → 启动
从最高点回撤 10% → 卖出 30%
```

**不要启用：** 止损卖、日涨跌幅卖、开板卖

指数 ETF 适合长期持有，不适合频繁交易。

---

## 五、真正重要的要素

对于指数基金投资，关键是：

1. **选什么指数** — 沪深300、中证500、纳斯达克100等
2. **每月投入多少钱** — 现金流规划
3. **市场大跌时是否有额外资金加仓** — 做好仓位管理
4. **什么时候分批止盈** — 移动止盈策略

> 💡 对于流动性好的 ETF，买一买二这些盘口细节对长期收益影响非常小，不必过度关注。
